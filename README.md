# 🌍 Problémy EU a USA s migranty

> Informační web o migračních tocích, politických výzvách a přístupech k jejich řešení v Evropské unii a Spojených státech amerických.

---

## 📋 O projektu

Tento projekt se věnuje problematice migrace – jednomu z nejspornějších témat současné světové politiky. Stránka přehledně srovnává situaci ve dvou největších demokratických celcích světa a nabízí věcný pohled na ekonomické, politické i humanitární aspekty migrace.

Web je navržen ve stylu novinového reportáže s důrazem na přehlednost a čitelnost.

---

## 📌 Obsah webu

### 🇪🇺 Evropská unie
- **Ekonomické dopady** – přínosy i náklady pracovní migrace
- **Nejednotnost členských států** – selhání Dublinského systému, Visegrádská čtyřka
- **Středomořské trasy** – nebezpečné přechody, role agentury Frontex
- **Sociální a kulturní výzvy** – integrace, identita, demografická situace

### 🇺🇸 Spojené státy americké
- **Jižní hranice** – migrační dynamika, původ migrantů
- **Politická polarizace** – střet Demokratů a Republikánů, DACA, hraniční zeď
- **Lidská práva** – podmínky v záchytných zařízeních, azylová řízení
- **Ekonomické dopady** – role migrantů na trhu práce

### 📊 Srovnávací tabulka
Přehledné porovnání přístupu EU a USA v klíčových oblastech: azylový systém, pohraniční kontrola, politické dělení a ekonomický dopad.

## 🚀 Jak zobrazit web

### Lokálně
1. Stáhněte soubor `index.html`
2. Otevřete ho v jakémkoli webovém prohlížeči (Chrome, Firefox, Edge…)

### Online – GitHub Pages
1. Nahrajte `index.html` do svého GitHub repozitáře
2. Přejděte do **Settings → Pages**
3. Jako zdroj vyberte větev `main` a složku `/ (root)`
4. Web bude dostupný na adrese `https://1lukami1.github.io/Problematika-EU-a-USA/`
