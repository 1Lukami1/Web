# Problémy EU a USA s migranty

Tento projekt se věnuje problematice migrace a výzvám, kterým čelí Evropská unie a Spojené státy americké. Stránka obsahuje přehled o migračních tocích, politických problémech a různorodých přístupech k jejich řešení.

## Obsah webu

- **Evropská unie (EU)**: 
  - Ekonomické, politické a sociální dopady migrace.
  - Výzvy spojené s řízením migračních toků.
  - Nejednotné politiky mezi členskými státy a jejich vliv na řešení.
- **Spojené státy americké (USA)**: 
  - Migrace na jižní hranici a její dynamika.
  - Politické spory o bezpečnost hranic a lidská práva migrantů.
  - Dopady migračních vln na ekonomiku a společnost.

## Jak zobrazit web

1. Otevřete `index.html` ve webovém prohlížeči.
2. Nebo nastavte projekt na **GitHub Pages** pro online přístup.
